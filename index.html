안녕하세요, 데이터 분석 엔지니어 김진희입니다.

이 포트폴리오는 저의 경력, 프로젝트, 그리고 기술 역량을 소개하기 위해 제작되었습니다.

📄 이력 요약

  • 이커머스 기반 데이터 분석 및 자동화 경험
  • MySQL, BigQuery, Python을 활용한 분석 업무 수행
  • GA4, GTM 기반 고객 행동 분석 및 리포트 구축

📊 주요 프로젝트

  • RFM 기반 고객 세그먼트 분석 - 구매 행동 기반 고객 분류
  • GA4 + BigQuery 퍼널 분석 - 유입부터 구매까지의 전환 퍼널 시각화
  • Cafe24 API 자동화 - 고객/주문 정보 수집 파이프라인 구축
